Luxury Vacation Listings & Services

Effortless Management Services offers luxury vacation rentals and a concierge type services. The site allows guests to view properties, check out availability, add additional services ranging from private drivers, chefs, yoga & physical trainers. The site allows users to book the property, services & pay all in one easy clean process. Includes seem-less social media and blog integration.

Project Information

  • Client:Effortless
  • Built With:HTML5 - Javascript - CSS3 - Sass - mySQL - Express - nodeJs - React - Passport - Squelize
  • Demo: www.effortlesshost.com
